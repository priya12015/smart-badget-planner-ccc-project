# smart-badget-planner

Report link: https://drive.google.com/drive/folders/1ohGFC3cxztrP5pLiAOMGaxQBY5E6luWF


Smart Budget Planner is a web app that uses the 0/1 Knapsack Dynamic Programming algorithm to help users choose the best combination of items within a budget. It takes item cost and value as input and outputs the optimal selection that maximizes total value efficiently.
